Build a full-stack multi-tenant QR code restaurant menu SaaS platform using Next.js, React, TailwindCSS, Node.js (Express) and PostgreSQL. The application must support multiple restaurants where each restaurant has its own menu, categories and items. Create a responsive mobile-first menu interface similar to a modern food delivery app with a sticky header containing the restaurant logo and a cart icon, a search bar with placeholder text “Search dishes, cuisines…” and a veg-mode toggle filter. Below the search bar implement a horizontally scrollable category carousel with circular category images and labels. Below categories add a featured “Only For You” section with a large image card and favorite icon. Under this section render menu item cards vertically showing item image, item name, price in INR, description and an orange “Add +” button that adds items to cart and updates quantity. Implement cart state management and a checkout summary panel. Create backend REST APIs for restaurants, categories, menu items, cart operations and orders. Implement database tables including users, restaurants, categories, menu_items, orders and order_items with proper foreign key relationships so each restaurant only accesses its own data. Add QR code generation so each restaurant has a unique menu URL like /menu/{restaurant_slug}. Implement veg filtering, search functionality and category scrolling. Add admin dashboard where restaurant owners can login and manage categories, menu items, prices and availability. Use JWT authentication for restaurant owners and secure all admin APIs. Include image upload support for menu items using cloud storage and store URLs in the database. Implement error handling, API validation and rate limiting. Ensure the project has modular folder structure including components, pages, api routes, services and database models. The UI must be modern with rounded cards, shadows, smooth hover effects and optimized image loading. Provide clean reusable components such as Header, SearchBar, CategoryCarousel, FeaturedCard, MenuItemCard and CartDrawer. Ensure all API calls fetch data dynamically from the database and render the UI accordingly.
